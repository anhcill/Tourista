# Tourista

Đây là ứng dụng quản lý đặt tour và khách sạn.

## Cấu trúc dự án
- `backend/`: Chứa mã nguồn Spring Boot.
- `frontend/`: Chứa mã nguồn Next.js.
- `database/`: Chứa các file schema và dữ liệu SQL.
- `demo/`: Thử nghiệm/demo (nếu có).

## Hướng dẫn chạy dự án

### Backend
1. Yêu cầu cài đặt Java và Maven.
2. Mở terminal, vào thư mục `backend/`.
3. Chạy lệnh: `./mvnw spring-boot:run` (hoặc chạy qua IDE).

### Frontend
1. Yêu cầu Node.js và npm.
2. Mở terminal, vào thư mục `frontend/`.
3. Chạy lệnh: `npm install` để cài đặt dependencies.
4. Chạy lệnh: `npm run dev` để chạy giao diện.
